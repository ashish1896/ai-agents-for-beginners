# Contributing to AI Agents for Beginners

This repository is a personal fork of [microsoft/ai-agents-for-beginners](https://github.com/microsoft/ai-agents-for-beginners) used for learning and tracking progress through the AI Agents course.

## How to Contribute

This fork is primarily for personal learning. However, if you find bugs or want to suggest improvements to the original course content, please contribute directly to the [upstream repository](https://github.com/microsoft/ai-agents-for-beginners).

## Reporting Issues

- For course content issues: Report on [microsoft/ai-agents-for-beginners](https://github.com/microsoft/ai-agents-for-beginners/issues)
- For fork-specific issues: Open an issue here

## Pull Requests

- Ensure your PR addresses a specific issue or improvement
- Follow the existing code style and conventions
- Keep PRs focused and atomic

## Learning Notes

Personal learning notes and progress tracking are maintained in `LEARNING_NOTES.md`.
